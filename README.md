Installation process for the project

Step 1:
    pip install django
Step 2:  
    django-admin startproject mobiz
    cd mobiz
Step 3:
    python manage.py runserver
Step 4:
    python manage.py startapp user
Step 5:
    python manage.py makemigrations
    python manage.py migrate
Step 6: 
    python manage.py runserver
